## Project Overview

This is a Next.js 16 personal website featuring:

- **Multi-language support** (English, Spanish, French)
- **Password protection** for site access
- **Dynamic content** across multiple pages
- **Contact form** with n8n integration
- **Responsive design** with Tailwind CSS

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```
